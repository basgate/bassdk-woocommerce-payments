== Description ==

Welcome to the official Basgate Payment Gateway plugin for Woocommerce.

* Wordpress v3.9.2 or higher
* Woocommerce v2.4 or higher
* PHP v7.4.0 or higher
* Php-curl

== Getting Started == 

New to BasgatePG? Use this [link](https://web.basgate.com:9191/) to create your Basgate for Business account and get access to exciting offers.

Before enabling the Basgate Payment Gateway on Woocommerce, make sure you have a registered business account with Basgate. Please visit - 
[Basgate Dashboard](https://web.basgate.com:9191/) to sign-up

== Step-1: Generate your API keys with Basgate ==

To generate the API Key,
* Log into your [Dashboard](https://web.basgate.com:9191/).
* Select the API Keys under Developers on the left menu-bar.
* Select the mode for which you want to generate the API Key from the menu.
* Click Generate now to generate a key for the test mode and in case of live mode, first activate the account by submitting documents and then generate the key by clicking the Generate now button.
* You will get the merchant ID and merchant key in response to the above. Please make a note of these to be used further.

Note: You have to generate separate API Keys for the test and live modes. No money is deducted from your account in test mode.
MID and merchant keys generation may take few minutes. In case you do not see these details, please logout and login after 5 minutes. Proceed now to generate these keys.

== Step-2: Plugin Installation == 

There are 2 ways of installing the Basgate payment gateway plugin:-
i)  Download the plugin repository from <here> 
            OR
ii) Install the plugin directly from the Wordpress dashboard

Note: In case you have installed the plugin directly from the wordpress dashboard, skip to Step-3. In case you have downloaded the repository from here, follow the steps below to complete the installation.

== Steps after downloading the plugin ==

1. Unzip and open the downloaded folder.
2. Copy the Basgate-payments folder from the unzipped folder.
3. Paste it into /wp-content/plugins/ directory or you may choose to upload the Basgate folder via the Woocommerce Webstore Admin panel.

== Step-3: Configuration ==

* Log into your WordPress admin and activate the Basgate plugin in WordPress Plugin Manager.
* Log into your WooCommerce Webstore account, navigate to Settings and click the Checkout/Payment Gateways tab
* Scroll down to the Checkout page and go to the setting option of Basgate under Gateway Display
* Click on Basgate to edit the settings. If you do not see Basgate in the list at the top of the screen make sure you have activated the plugin in the WordPress Plugin Manager
* Fill in the following credentials.
	* Enable - Enable check box
	* Title - Basgate
	* Description - Default
	* Merchant Identifier - Staging/Production MID provided by Basgate
	* Secret Key - Staging/Production Key provided by Basgate
	* Website Name - Provided by Basgate
	* Industry Type - Provided by Basgate
	* Environment - Select environment type

Your Basgate payment gateway is enabled. Now you can accept payment through Basgate.
In case of any issues with integration, please [get in touch](https://basgate.com).

== Installation ==

* Download Basgate Payment Gateway plugin 
* Upload all plugin files in "wp-content/plugins/" directory
* Install and activate the plugin from Wordpress admin panel
* Visit the WooCommerce > Settings page to configure Basgate Payment Gateway Plugin.
* Your Basgate Payment Gateway plugin is now setup. You can now accept payments through Basgatem.
